Welcome to Object Detection web app!

This web-app helps you to detect the objects found in the house. It uses an object detection model called CoCoSSD that aims to identify multiple objects in a single image. You can simply select the room and detect multiple objects found in that particular room. It is actually kinda useful in case you forgot where you kept your things, you can just simply check where the object is using our object detection web app.
